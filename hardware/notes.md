# 硬件复现注意事项

公开资料显示，sparkbot-45coll 是在乐鑫 ESP-SparkBot 基础上修改，增加无线充电、小车巡线等功能，并优化供电、小车电池给头部充电、一键开关机、10 秒无运动自动关机、四个触摸按键等。

复现时重点检查：

1. 头部主控：ESP32-S3-WROOM-1-N16R8；注意公开说明中提到最终使用有板载天线版本。
2. 小车主控：ESP32-C3。
3. 屏幕：常见为 ST7789 240×240 LCD。
4. 音频：I2S 麦克风/功放/扬声器链路。
5. 摄像头：OV2640 / DVP 摄像头链路。
6. IMU：公开资料中原 ESP-SparkBot 使用 BMI270，45coll README 提到发布固件还区分 BMI220 / BMI270。
7. 供电：电池、USB、充电、无线充电与头部/底盘供电路径要分清。
8. 小车控制：网页控制、巡线、AI 控制小车动作是面试亮点。
